# Stolin Edition 3.0

**Autorski mod do Mindustry**, który wprowadza zupełnie nową planetę: **Kalistonarix**.  
Stworzony przez **Jankowo119**.

## Co zawiera

- Własna planeta z unikalnymi sektorami
- Nowe bloki, generatory i drzewko technologiczne
- Niestandardowa atmosfera i siatka
- Oryginalna mechanika rozgrywki i progresji
- 'Ważne'Sprite nie jest dopracowany no nie jestem w tym dobry!

## Instalacja

1. Pobierz plik `.zip` z modem
2. Otwórz Mindustry
3. Przejdź do **Mods** → **Import Mod**
4. Wybierz pobrany plik `.zip`

## Wymagania

- Mindustry w wersji V8 wersja od 154 do 155 jak chcesz to zmień "minGameVersion": na wcześniejszą
wersję
